# JSON Formatter (json-lab)

브라우저에서 JSON을 붙여넣거나 업로드하면 곧바로 포맷팅하고 유효성을 알려주는 정적 웹앱입니다. JSON을 자주 확인하는 개발자를 위해 빠른 포맷·검증·공유 흐름에 초점을 맞추었습니다.

- 라이브 데모: https://ilestri.github.io/json-lab/
- 현재 버전: v2

## 무엇을 할 수 있나요?
- JSON 포맷/Minify, 들여쓰기(2/4/tab)·키 정렬 선택, 에러 라인/컬럼 표시
- 텍스트 입력, `.json` 파일 선택·드래그&드롭, 클립보드 읽기, 샘플 불러오기
- 결과 복사, 줄 번호 있는 코드 블록, 상태바 메시지/가이드, 토스트 알림
- 최근 스니펫 5개 저장/불러오기, LZ 압축 공유 링크(URL 파라미터)
- 도구 탭: JSON Schema 검증(Ajv), 두 JSON Diff, 트리 뷰 탐색, URL에서 JSON fetch
- 설정 탭: 포맷 기본값(자동 포맷/정렬/Minify/업로드·Fetch 자동 포맷)과 가독성(글자 크기/줄 간격/대비), 라이트·다크 테마. 모두 LocalStorage 자동 저장.

## 이렇게 사용해 보세요
1. 메인 페이지에서 JSON을 붙여넣거나 `.json` 파일을 드래그&드롭합니다.
2. `포맷팅`을 누르거나 자동 포맷이 켜져 있다면 즉시 결과와 상태를 확인합니다.
3. 필요한 경우 `Minify`, `키 정렬`, `공유 링크`, `최근 스니펫` 기능을 활용해 저장하거나 공유합니다.
4. 더 정교한 작업이 필요하면 상단 탭에서 Schema/Diff/Tree/Fetch 도구를 열어 사용하세요.
5. 본인에게 편한 들여쓰기·테마·글자 크기 등은 `설정` 탭에서 바꾸면 다음 방문에도 유지됩니다.

## 유용한 팁
- 단축키: `Cmd/Ctrl + Enter` 포맷, `Cmd/Ctrl + Shift + C` 결과 복사, `Alt + M` Minify 포맷.
- 파일·URL 로딩 시 자동 포맷 옵션을 켜두면 결과를 바로 볼 수 있습니다.
- 공유 링크는 입력 JSON을 압축해 URL에 담으므로 간단한 샘플 공유에 적합합니다.
